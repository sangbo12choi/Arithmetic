# 사칙연산 모듈 테스트 프로젝트

## 프로젝트 개요
사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈) 기능의 정확도와 예외 처리를 검증하는 테스트 프로젝트입니다.

## 테스트 정보
- **테스트 ID**: TC-CMM-001 (Common Module) / TC-AO-001 (Arithmetic Operations)
- **테스트 목적**: 사칙연산 정확도 테스트
- **테스트 범위**: 공통 모듈
- **작성일**: 2020-09-01
- **버전**: v1.0
- **작성자**: 홍길동
- **승인자**: 박문수

## 개발 방법론
본 프로젝트는 **TDD (Test-Driven Development)** 방식으로 진행됩니다:
1. **RED**: 실패하는 테스트 작성
2. **GREEN**: 테스트를 통과하는 최소한의 코드 작성
3. **REFACTOR**: 코드 리팩토링 및 개선

## 테스트 환경
- **언어**: Python
- **IDE**: PyCharm
- **운영 체제**: Windows 10

## 테스트 케이스

### 기본 사칙연산 테스트

| 테스트 케이스 | 입력값 | 예상값 | 중요도 | 상태 |
|--------------|--------|--------|--------|------|
| 덧셈 (양수) | 1 + 10 | 11 | 중요 | 성공 |
| 덧셈 (0 포함) | 0 + 1 | 1 | 중요 | 성공 |
| 덧셈 (음수) | -1 + (-10) | -11 | 보통 | 성공 |
| 뺄셈 | 5 - 2 | 3 | 중요 | 성공 |
| 곱셈 (음수) | -5 * -3 | 15 | 보통 | 성공 |
| 곱셈 (0 포함) | 0 * 10 | 0 | 낮음 | 성공 |
| 정수 나눗셈 | 5 / 2 | 2 | 중요 | 성공 |
| 소수점 나눗셈 (quotient) | 5 ÷ 2 | 2.5 | 보통 | 성공 |
| 나눗셈 (음수) | -10 / 2 | -5 | 중요 | 성공 |

### 예외 처리 테스트

| 테스트 케이스 | 입력값 | 예상값 | 중요도 | 상태 |
|--------------|--------|--------|--------|------|
| 0으로 나누기 | 0 / 0 | ArithmeticException 예외 발생 | 중요 | 성공 |

## 실행 방법

### 테스트 실행
```bash
# pytest를 사용한 테스트 실행
pytest tests/

# 상세 출력과 함께 실행
pytest tests/ -v

# 커버리지 리포트 생성
pytest tests/ --cov=src --cov-report=html
```

## 참고사항
이 테스트 케이스는 기능의 정확성과 예외 처리를 포함하여 클래스의 모든 기능을 검증합니다.
다양한 조합(양수, 음수, 0)을 포함한 테스트 케이스를 통해 연산의 정확도와 범위를 검증합니다.
